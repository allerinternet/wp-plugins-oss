== Description ==

Aller GA is used to inject the javascript code for Google Analytics in the <head> tag on the site. User of the plugin shall enable the function in preferences. Entering the personal UA-number is also required.

To use this plugin a Google Analytics account is required!


== Installation ==

Upload aller-ga.php to the /wp-content/plugins/ directory.
